Auto Gifted Basic Bee Tool (BETA)

This tool is in early beta and may be buggy. It is recommended to watch the tool run to make sure it doesn't do anything unexpected. If you encounter any issues, please report them on the discord server.


When you press start, the tool focuses Roblox, waits a short countdown (set below, default of 3 seconds), captures your current mouse position as the bee slot, then alternates Basic Egg, Royal Jelly, Basic Egg, and repeats until it finds a Gifted Basic Bee or a paused result.

To stop this tool, use the macro stop hotkey: F3. You can change this in Config.

Please note: this is beta, it may mess up, it is not garunteed to work, please watch it run and report any issues you see on the discord. The tool relies on OCR to read the result of each roll, so it may misread sometimes, especially if your game resolution is low or if you have a lot of screen clutter. If it misreads a result, it may do extra rolls or stop when it shouldn't, so it's important to keep an eye on it while it's running.
